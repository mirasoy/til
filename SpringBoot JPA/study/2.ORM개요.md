# ORM개요

- ORM이란
    - Object -Relaction Maping
    - 매핑 정보를 기술한 메타데이터를 사용하여 자바 애플리케이션 객체를 SQL 데이터베이스에 자동으로(깔끔하게) 영속화해주는 기술
    - 장점 :  생산성 / 유지보수성 / 성능 / 벤더독립성(DB에 구애받지않음)
    - 단점  : 학습비용

- 도메인 기반으로 코딩을 하려는 이유
    - 더 객체지향적인 프로그래밍 가능
    - 각종 디자인패턴 사용 가능
    - 코드 재사용이 가능
    - 비즈니스 로직 구현이 편함
    - 테스트가 용이
    
     > 생산성, 유지보수성, 성능, 벤더 독립성
